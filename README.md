# CLI Minesweeper written in Ada 95

![Image of program](./example.png)

## Compile and Run (Linux)

```shell
sudo apt install gnat
make -C src
./mine
```

## Controls
<table>
    <tr>
        <th><kbd>w</kbd>, <kbd>a</kbd>, <kbd>s</kbd>, <kbd>d</kbd></th>
        <th>Move cursor</th>
    </tr>
     <tr>
        <th><kbd>SPACE</kbd></th>
        <th>Open cell</th>
    </tr>
     <tr>
        <th><kbd>f</kbd></th>
        <th>Flag cell</th>
    </tr>
     <tr>
        <th><kbd>q</kbd></th>

    </tr>
</table>
